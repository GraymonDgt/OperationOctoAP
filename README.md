# Operation Octo AP Setup

BEFORE SETTING THIS UP, CREATE A NEW FILE IF YOU DONT WANT TO LOSE YOUR EXISTING PROGRESS! THE CURRENT FILE WILL BE OVERWRITTEN UPON STARTUP

Download melonloader here: https://melonloader.co/download.html

Click on Operation Octo and install the latest verison

Launch the game then close it

On steam, goto Operation Octo's settings -> Manage -> Browse local files

Extract OperationOctoArchipelago.zip so that config.json is in the "Mods" folder and Archipelago.Multiclient.Net.dll is in "UserLibs"

Edit config.json to change slot name/port number/server

Launch the game and it will connect on startup

You will need the archipelago text client for hints/ logs

Also hold space to play at 2x speed
